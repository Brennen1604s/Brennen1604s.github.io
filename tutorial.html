HTML

You can think of HTML as the skeleton for your webpage. It creates the basics of your website without the style. HTML is written with tags. The basic tags are:

You can edit what text each tag displays by changing the text in between the "arrows". For example, if you want to change the text in the first paragraph, edit the text in between the "p" tags.

Now that we've covered the basics of HTML, lets move to CSS!

CSS

CSS, or Cascading Style Sheets, is the design of your webpage. If you want to add a picture to your website, you use css. To add css to a tag, you simply add 'style=" "' to the specific tag. Within the style, you can change things like text alignment, font size, font weight, what font you're using, the height of the lines, and how many pixels you want the border to be.

For full understanding of what you can do with css, visit w3schools.
