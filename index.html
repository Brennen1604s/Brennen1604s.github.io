Welcome to Kreate

What is Kreate?

Kreate is a fully customizable website creation tool that can be used by anyone! Here at Kreate, we offer free and paid-for website templates. You can access these templates by going into the store dropdown menu at the top of your screen and clicking either free templates or website templates.

How to use Kreate

When using Kreate, it is recommended that you have a basic knowledge of HTML and CSS, but if you don't, that's fine too! Just click the tutorial button at the top of your screen to receive help. Now that we've covered the basics, let's dive further in. Once you download the files from Kreate, you can enter them using a text editor. In that file, you can change any of the HTML and CSS code to fit your design and functionality needs!

HTML and CSS

HTML and CSS are your best friends when creating a website. Luckily for you, we created the website for you! All you have to do is customize it. For some people, learning HTML and CSS can be difficult. For others, it can be a breeze. When you purchase and install one of our website files, most of its instructions will be included in the html files. Each download will be equiped with a README text file that will include information on the project. More information will be included in the HTML and CSS files as comments. Comments will be colored in gray. If these instructions do not provide enough help, you can visit the w3schools website for more assistance.
